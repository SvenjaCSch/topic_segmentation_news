# topic_segmentation_news
Topic Segmentation of news data in english and german with Tkinter Interface

Tkinter Interface which uses an youtube link as an input and:
- download audiofile
- making a speaker diarization with transcript via automatic speech recognition
- alternativ with automatic speech recognition and segmentate the tet via texttiling
- categorize data with Markov, Snorkel and BERTopic
As an output you get an table of the segmented and transcripted news data with the fitting categories

As training data for the categorizing models BBC news data and Ten Thousand German News Articles are used.
